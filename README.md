# 🎮 Plataforma de Gamificação para Aprendizagem

## 📌 Descrição

Este projeto tem como objetivo desenvolver uma plataforma digital de gamificação baseada em quizzes para auxiliar estudantes na revisão de conteúdos acadêmicos.

A aplicação utiliza mecânicas de jogos como pontuação (XP), níveis, metas diárias e feedback imediato para aumentar o engajamento e melhorar a retenção do aprendizado.

---

## 🚀 Funcionalidades

* ✅ Sistema de quizzes interativos
* ✅ Feedback em tempo real
* ✅ Sistema de XP e níveis
* ✅ Metas diárias e streaks
* ✅ Trilhas de aprendizagem
* ✅ Dashboard de desempenho

---

## 🧠 Objetivo Acadêmico

Promover o aumento do engajamento dos alunos e melhorar o desempenho acadêmico, com meta de aumento de 20% na taxa de acertos após uso contínuo da plataforma.

---

## 🛠️ Tecnologias Utilizadas

### Frontend

* React + Vite
* TailwindCSS

### Backend

* Node.js
* Express

### Banco de Dados

* PostgreSQL

---

## ⚙️ Como rodar o projeto

### 1. Clonar repositório

```bash
git clone https://github.com/seu-usuario/gamified-learning-platform.git
```

### 2. Backend

```bash
cd backend
npm install
npm run dev
```

### 3. Frontend

```bash
cd frontend
npm install
npm run dev
```
